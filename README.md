# Heart Disease Prediction using Machine Learning

This project uses Machine Learning to predict the presence of heart disease based on patient health-related features.

## Project Overview

The goal of this project is to build a classification model that can predict whether a person has heart disease or not. The project includes data exploration, preprocessing, model training, and model evaluation.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Support Vector Machine (SVM)

## Dataset

The dataset contains medical attributes related to heart health, such as age, cholesterol, blood pressure, chest pain type, maximum heart rate, and other clinical features.

## Project Workflow

1. Loaded the heart disease dataset
2. Performed exploratory data analysis (EDA)
3. Checked missing values and duplicates
4. Encoded categorical features
5. Scaled numerical features using StandardScaler
6. Split the dataset into training and testing sets
7. Trained a Support Vector Machine (SVM) model
8. Evaluated the model using accuracy, confusion matrix, classification report, and ROC curve

## Model Used

- Support Vector Machine (SVM)

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- ROC Curve
- AUC Score
